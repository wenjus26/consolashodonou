# Consolas HODONOU - Portfolio Premium "Wax Tech Empower"

Ce dépôt contient le code source du portfolio professionnel de **Consolas HODONOU**, Data Scientist & Biostatisticienne.

## 🚀 Vision du Projet
Un portfolio "Awwwards-level" conçu avec une esthétique minimaliste en 3 couleurs, mettant en avant l'alliance entre la rigueur scientifique (Biostatistiques) et l'innovation technologique (IA).

## 🛠 Tech Stack
- **Structure** : HTML5 sémantique
- **Styling** : Tailwind CSS (Custom Theme)
- **Interactions** : Alpine.js
- **Animations** : GSAP (ScrollTrigger) & FontAwesome 6
- **Data Viz** : Chart.js (Radar Skills Chart)
- **Backend Deployment** : Netlify (Contact Form & Hosting)

## ✨ Fonctionnalités Clés
- **Hero Section Premium** : Encadrement Glassmorphism avec tags flottants dynamiques.
- **Python Terminal Animé** : Simulation de code en direct pour le déploiement de modèles.
- **Timeline Senior** : Trajectoire professionnelle de 2018 à 2025.
- **AI Chatbot Widget** : Interface conversationnelle entraînée sur le profil.
- **Magnetic Cursor** : Navigation interactive fluide.
